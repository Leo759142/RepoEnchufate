# Operaciones Diarias

Guía de procedimientos operativos diarios para el personal de Enchufate.

## Apertura del Local (7:00 AM)

### Checklist de Apertura

- [ ] **Llegar 15 minutos antes**
- [ ] Desactivar alarma de seguridad
- [ ] Encender luces y aire acondicionado
- [ ] Encender sistema de kiosk en recepción
- [ ] Verificar cámara CCTV funcionando
- [ ] Revisar reservas del día en sistema
- [ ] Revisar notificaciones/mensajes
- [ ] Preparar área de recepción
- [ ] Llenar dispensador de agua
- [ ] Conectar computadora de recepción
- [ ] Sintonizar televisor de área común

### Verificaciones Técnicas

1. **Kiosk Manager**
   - ¿Sistema respondiendo?
   - ¿Base de datos sincronizada?
   - ¿Impresora funcionando?

2. **Cubículos**
   - Verificar visualmente
   - Comprobar que están limpios
   - Revisar estatus en sistema

3. **Conectividad**
   - Ping a API
   - Test de velocidad internet
   - Revisar status de WiFi

### Preparación de Servicios

```
7:00 AM - Abrir puertas
7:15 AM - Sistema listo para clientes
7:30 AM - Área común operacional
```

## Durante el Horario de Operación

### Tareas Cada Hora

- [ ] Revisar ocupación de cubículos
- [ ] Verificar alarmas/alertas
- [ ] Comprobar CCTV
- [ ] Revisar chat de soporte
- [ ] Caminar por instalaciones (cada 2 horas)

### Tareas Cada 4 Horas

- [ ] Revisar cafetería (reabastecer si es necesario)
- [ ] Limpiar baños
- [ ] Vaciar papeleras
- [ ] Verificar temperatura
- [ ] Revisar histórico de eventos

### Atención al Cliente

**Protocolo de Recepción**:

1. Saludar amablemente
2. Verificar si es cliente recurrente
3. Ofrecer opciones de cubículo
4. Explicar tarifas y servicios
5. Processar pago/membresía
6. Asignar cubículo
7. Proporcionar credenciales
8. Explicar normas básicas
9. Resolver dudas
10. Desear buena sesión

### Gestión de Problemas

**Problema en Cubículo**:
1. Acudir inmediatamente
2. Escuchar al cliente
3. Intentar resolver (reinicio, etc.)
4. Si no se resuelve: Cambiar a otro cubículo
5. Documentar incidente
6. Contactar técnico si es necesario

### Tareas Prioritarias

- [ ] Maximizar ocupación

- [ ] Atención rápida a clientes
- [ ] Soporte técnico inmediato
- [ ] Mantener orden y limpieza

## Cierre del Local (11:00 PM)

### Checklist de Cierre

- [ ] **30 min antes**: Anunciar cierre
- [ ] **15 min antes**: Dar aviso final
- [ ] A las 11:00 PM: No permitir nuevos clientes
- [ ] Cubículos activos: Dar 5 min para terminar
- [ ] Finalizar todas las sesiones activas
- [ ] Generar reporte de cierre
- [ ] Cobrar pendientes
- [ ] Recopilar reportes de día
- [ ] Apagar computadoras
- [ ] Limpiar área común
- [ ] Apagar cafetería
- [ ] Cerrar baños
- [ ] Apagar luces
- [ ] Verificar CCTV activo
- [ ] Cerrar puertas
- [ ] Activar alarma
- [ ] Dejar reporte para gerente del día siguiente

### Reporte de Cierre

Incluir:
- Número de sesiones del día
- Ingresos totales
- Incidentes reportados
- Mantenimiento realizado
- Inventario de consumibles
- Observaciones importantes

**Guardar en**: `/reports/cierre_[YYYYMMDD].pdf`

## Sistema de Alertas

### Alertas que Requieren Acción Inmediata

1. **Falla de Sistema**
   - Contactar al DBA
   - Usar backup manual
   - Registrar incidente

2. **Falla de Internet**
   - Verificar conexión ISP
   - Contactar ISP si aplica
   - Usar modo offline

3. **Problema de Seguridad**
   - Desactivar área afectada
   - Contactar seguridad
   - Documentar incidente

4. **Daño a Hardware**
   - Aislar equipo
   - Marcar como "no usar"
   - Contactar técnico

## Procedimientos de Emergencia

### Incendio

1. Evacuar inmediatamente
2. Llamar a bomberos (911)
3. Reunir en punto seguro
4. Apagar sistemas (si es seguro)

### Falla de Corriente

1. Activar generador de emergencia
2. Anunciar a clientes
3. Ofrecer compensación por tiempo perdido
4. Contactar electricista

### Fallo de Agua

1. Cerrar suministro principal
2. Avisar a clientes
3. Cerrar baños
4. Contactar fontanería

### Intrusión/Seguridad

1. Llamar a policía (911)
2. No confrontar al intruso
3. Evacuar si es necesario
4. Contactar gerente

# Seguridad y Privacidad

Información importante sobre seguridad y protección de tu información personal.

## Seguridad Física

### Vigilancia y Protección

Enchufate cuenta con:

- **Sistema de CCTV 24/7**: Cámaras de vigilancia de 4K en todas las áreas
- **Personal de seguridad**: Disponible en horarios pico
- **Control de acceso**: Entrada y salida registrada
- **Cerraduras de seguridad**: En todas las áreas sensibles
- **Botón de emergencia**: En cada cubículo

### Protección de Pertenencias

::: warning Importante
Enchufate no es responsable de artículos personales olvidados o robados.
:::

**Recomendaciones**:
- Retira tus pertenencias personales al salir
- No dejes computadoras portátiles o dispositivos personales sin vigilancia
- Usa los casilleros seguros disponibles (bajo solicitud)
- Cierra tu sesión antes de abandonar el cubículo

## Seguridad Digital

### Protección de Tu Computadora

Todas nuestras máquinas incluyen:

- **Antivirus actualizado**: ESET NOD32 + Windows Defender
- **Firewall activo**: Windows Defender Firewall
- **Actualizaciones de seguridad**: Aplicadas diariamente
- **Software contra malware**: Malwarebytes Premium
- **VPN opcional**: Disponible para conexiones seguras

### Buenas Prácticas de Seguridad

::: tip Consejo de Seguridad
Sigue estos pasos para mantener tu información segura:
:::

1. **Cierra sesión siempre**
   - Cierra todos los navegadores
   - Cierra sesión en correo y redes sociales
   - Cierra la computadora correctamente

2. **No guardes información sensible**
   - No guardes contraseñas en el navegador
   - No dejes datos bancarios guardados
   - No guardes números de tarjetas de crédito

3. **Usa navegación privada**
   - Para operaciones bancarias
   - Para compras online
   - Para cuentas de email

4. **Evita redes públicas**
   - Nuestro wifi es seguro, pero aún así:
   - No accedas a banca online desde wifi público
   - No hagas transacciones sensibles en wifi abierto

Al registrarte, recopilamos:

Tienes derecho a:

Contacta a privacidad@enchufate.com para solicitudes de GDPR.

## Limpieza de Computadoras

### Proceso de Limpieza

Después de cada sesión:

1. Se ejecuta un script de limpieza automático
2. Se eliminan archivos temporales
3. Se vacía la papelera de reciclaje
4. Se limpian caché del navegador
5. Se reinicia el sistema

**Esto ocurre de forma automática - no necesitas hacer nada.**

### Tus Archivos Personales

::: danger Advertencia
**Todos los archivos se eliminan después de tu sesión.**
:::

Para guardar tus archivos:
- Cópialos a USB
- Cárgalos a Google Drive, Dropbox o OneDrive
- Envíalos por correo
- Usa nuestro servicio de respaldo en nube

## Protección de Menores

### Políticas para Menores de Edad

**Menores de 16 años**:
- Deben estar acompañados de un adulto responsable
- O presentar autorización escrita de padres/tutores

**Control parental**:
- Acceso bloqueado a contenido inapropiado
- Restricción de navegación en sitios adultos
- Límites de tiempo configurables

**Seguridad**:
- Supervisión especial durante horarios escolares
- Reporte de actividad a padres (si está configurado)
